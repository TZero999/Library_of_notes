Данные команды используются для входа в текстовый редактор. К примеру:
$ nano newfile
Либо создаём новый файл с именем “newfile” и начинаем его редактирование в редакторе nano,
либо начнём редактирование существующего файла “newfile” (если он имеется) с помощью того же редактора.

Те же вещи применимы и для команды vi, которая открывает другой редактор под названием “vi”.

Редактор "vi" довольно своеобразен. 
Хороший гайд по его использованию тут:
https://docs.altlinux.org/ru-RU/archive/2.3/html-single/junior/alt-docs-extras-linuxnovice/ch02s10.html

- Важно: в чём разница?
Nano, в отличии от vi, не является стандартным редактором. Скорее всего для его использования 
необходимо будет установить его: 
$ sudo apt update && sudo apt install nano
