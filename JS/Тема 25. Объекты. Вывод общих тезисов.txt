ИТОГО:

1) Объекты – это ассоциативные массивы с рядом дополнительных возможностей.
Они хранят свойства (пары "ключ:значение")

Ключи должны быть строками или символами.
Значения могут иметь любой тип.

2) Чтобы получить доступ двумя способами:
- Запись через точку: obj.property.
- Запись через квадратные скобки: obj["property"]

ТОЛЬКО квадратные скобки [] позволяют взять КЛЮЧ из ПЕРЕМЕННОЙ!

3) Дополнительные операторы:
Свойства можно удалять, изменять их значения, проверять их существование, перебирать в цикле:

- delete obj.property - удаление свойства
- obj.property = "Value" - изменение значения свойства
- "key" in obj - проверка существования свойства
- for (let key in obj) - перебор свойств объекта через цикл

4) Виды объектов.
То, что мы прошли в этой главе - простые оюъекты.

Но в JS есть много других объектов:
"Array" - для хранения упорядоченных коллекций данных,
"Date" - для хранения информации о дате и времени,
"Error" - для хранения информации об ошибках,
и т.д.

Порой в беседе их называют типами данных, но формально они ими не являются, это все объекты.
