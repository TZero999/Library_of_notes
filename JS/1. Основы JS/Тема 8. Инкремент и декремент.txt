Инкремент / Декремент:
Форма записи: ++ / --
Можно ПРИМЕНЯТЬ инкремент и декремент только к ПЕРЕМЕННЫМ!

Примеры:
let counter = 2;
counter++; // counter = 3.

НО:
let counter = 2++; // Здесь в консоль будет выдана ошибка, код умрёт.

Инкремент и декремент имеют 2 формы записи:

Постфиксная: Когда оператор идёт после переменной (counter++)
Префиксная: когда оператор идёт перед переменной (++counter)

Давайте проясним этот момент. Как мы знаем, все операторы возвращают значение. Операторы инкремента/декремента не исключение.
Префиксная форма возвращает новое значение, в то время как постфиксная форма возвращает старое (до увеличения/уменьшения числа).

Т.О.:
let counter = 1;
let a = ++counter; // (*)
alert(a); // 2

let counter = 1;
let a = counter++; // Здесь меняем ++counter на counter++
alert(a); // 1

В строке (*) постфиксная форма counter++ также увеличивает counter, но возвращает старое значение (которое было до увеличения). Так что alert покажет 1.

Как это использовать в быту?
Если результат оператора НЕ ИСПОЛЬЗУЕТСЯ, а нужно только увеличить/уменьшить переменную, тогда БЕЗ РАЗНИЦЫ, какую форму использовать.
Если результат надо сразу же использовать - пользуемся префиксной формой.
Если надо увеличить переменную, но вывести её значение ДО УВЕЛИЧЕНИЯ - пользуемся ПОСТФИКСНОЙ формой.
