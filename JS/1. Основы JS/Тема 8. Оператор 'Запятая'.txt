Оператор «запятая» (,) редко применяется и является одним из самых необычных.
Иногда он используется для написания более короткого кода, поэтому нам нужно знать его, чтобы понимать, что при этом происходит.

Оператор «запятая» предоставляет нам возможность вычислять несколько выражений, разделяя их запятой.
Каждое выражение выполняется, но возвращается результат только последнего.

Вот пример:

let a = (1 + 2, 3 + 4);

alert( a ); // 7 (результат вычисления 3 + 4)

Скобки здесь объясняются тем, что приоритет запятой очень мал. Если надо объяснение - смотри ниже:
Без них в a = 1 + 2, 3 + 4 сначала выполнится +, суммируя числа в a = 3, 7, затем оператор присваивания = присвоит a = 3.
То, что идёт дальше, будет игнорировано. Всё так же, как в (a = 1 + 2), 3 + 4.
