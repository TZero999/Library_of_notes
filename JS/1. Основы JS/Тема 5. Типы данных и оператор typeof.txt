Всего в JS 8 типов данных:

1) number - целочисленные значения и числа с плавающей запятой.
2) bigint - числа, большие чем (2^53 -1)/ меньшие, чем -(2^53 -1).
3) boolean - логический тип данных, возвращает только true/ false.
4) string - строка, заключается в кавычки.
5) symbol - уникальные идентификаторы в объектах.
6) undefined - неопределённое значение.
7) null - содержит только значение null.
8) object

Первые 7 - примитивные. 
Объекты - сложный тип данных.

TYPEOF

Оператор typeof возвращает тип аргумента. 

Есть 2 синтаксиса оператора typeof:

- typeof x - 
- typeof (function)

ПРИМЕРЫ:

typeof 0 // "number";

typeof null // "object" // Да, ошибка языка.

Вот как можно использовать typeof для отсеивания свойств объекта:

if(typeof (menu[key]) == 'number') // применяет блок if для числовых свойств объекта. ВНИМАНИЕ, number в кавычках!!!
