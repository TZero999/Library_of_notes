И «&&» находит первое ложное значение

Если имеем строку:
result = value1 && value2 && value3;

Оператор && выполняет следующие действия:
1) Вычисляет операнды слева направо.
2) Каждый операнд преобразует в логическое значение. Если результат false, останавливается и возвращает исходное значение этого операнда.
3) Если все операнды были истинными, возвращается последний.

Другими словами, И возвращает первое ложное значение. Или последнее, если ничего не найдено.
Выходит, что отличие И от ИЛИ лишь в том, что И возвращает первое ложное значение, а ИЛИ –  первое истинное.

Т.О. если передать несколько значений подряд:
И вернёт первое ложное значение - alert( 1 && 2 && null && 3 ); // null
Или же вернёт последнее значение, если все они верны - alert( 1 && 2 && 3 ); // 3

Приоритет оператора И && больше, чем ИЛИ ||, так что он выполняется раньше.

С помощью И тоже можно заменить оператор if:

let x = 1;
(x > 0) && alert( 'Greater than zero!' );

Аналогично следующему:

let x = 1;
if (x > 0) {
  alert( 'Greater than zero!' );
}

Очевидно, что вариант с if выглядит более громоздким, но читается намного лучше.
