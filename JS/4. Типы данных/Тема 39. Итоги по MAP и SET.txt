- MAP
MAP - коллекция пар ключ-значение.

Отличия от Object:
1) Ключ может иметь любой тип данных;
2) Имеет дополнительное свойство size.

Методы и свойства MAP:
1) new Map([iterable]) – создаёт коллекцию, можно указать перебираемый объект (обычно массив) из пар [ключ,значение] для инициализации.
2) map.set(key, value) – записывает по ключу key значение value.
3) map.get(key) – возвращает значение по ключу или undefined, если ключ key отсутствует.
4) map.has(key) – возвращает true, если ключ key присутствует в коллекции, иначе false.
5) map.delete(key) – удаляет элемент по ключу key.
6) map.clear() – очищает коллекцию от всех элементов.
7) map.size – возвращает текущее количество элементов.

- SET
SET - коллекция уникальных значений - "множество".

Методы и свойства SET:
1) new Set([iterable]) – создаёт Set, можно указать перебираемый объект со значениями для инициализации;
2) set.add(value) – добавляет значение (если оно есть - игнорируется). Возвращает тот же объект set;
3) set.delete(value) – удаляет значение. Возвращает true - при успехе, или false - если значения нет;
4) set.has(value) – проверяет наличие элемента. Возвращает true или false;
5) set.clear() – удаляет все имеющиеся значения;
6) set.size – возвращает количество элементов в множестве.

Перебор MAP и SET осуществляется в порядке добавления элементов.
Поэтому это упорядоченные коллекции, но менять порядок элементов или получить их по номеру - нельзя.
