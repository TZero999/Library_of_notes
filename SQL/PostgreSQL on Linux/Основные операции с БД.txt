Все нижеследующие команды выполняются через командруню строку PostgreSQL.
Входим в неё командной
- sudo -u tzero psql

1. Проверяем пользователей и БД:

Сначала список БД:
- \list

Затем список пользователей:
- \du

2. Создаём новую БД:

- create database family_test;
Вывод корректнйо команды: "CREATE DATABASE"

3. Создаём нового пользователя:

- create user author
Создаст нового пользователя с правами только на чтение данных из БД.

- \password author
Позволит задать пароль для нового пользователя

- create user author with login password 'qwerty'
Такая комбинация команд позволит сразу создать пользователя и указать пароль.

4. Удаление БД или пользователя:

- drop database family_test
Позволит удалить БД family_test

- drop user author
Позволит удалить пользователя tzero

5. Входим в существующую БД под существующим пользователем:

- psql family_test tzero;
Выполняем данную команду из терминала Linux.
