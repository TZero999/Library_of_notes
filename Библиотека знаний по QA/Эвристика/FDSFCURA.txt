"FDSFCURA" - это эвристика, представляющая собой перечень всех типов тестирования,
которые необходимо применять для проверки системы.

Она показывает, что существует огромное множество тестов, которыми мы можем проверить
наш продукт, для того, чтобы понять, что и как мы должны тестировать. 

Расшифровывается это следующим образом:

- Function Testing
Функциональное тестирование;

- Domain Testing
Тестирование доменных имен;

- Stress Testing
Стресс-тестирование;

- Flow Testing
Тестирование потоков;

- Scenario Testing
Тестирование сценариев;

- Claims Testing
Тестирование требований;

- User Testing
Пользовательское тестирование;

- Risk Testing
Тестирование рисков;

- Automatic Testing
Автоматизация.
