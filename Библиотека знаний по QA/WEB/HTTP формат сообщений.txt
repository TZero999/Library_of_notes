* Общая информация:
HTTP - Hyper Text Transfer Protocol

HTTP протокол действует на "Транспортном" уровне сетевых моделей.
Базируется на обмене сообщениями-данными между сервером и клиентом.

Существует 2 типа сообщений:
- Запросы
- Ответы

Сообщения HTTP состоят из текстовой информации в кодировке ASCII.

В HTTP 1.1 сообщения представляют собой обычный текст.
В HTTP 2 сообщение разделяется на фреймы, что позволяет оптимизировать и
повысить производительность обмена сообщениями.

* Структура HTTP-сообщения:
Запросы и ответы имеют одинаковую структуру:

1. Стратовая строка
Для запросов в ней хранится метод и версия протокола.
Для ответов - версия протокола и "статус-код".

2. Набор HTTP-заголовков
Заголовки определяют запрос и описывают тело сообщения.
Примеры заголовков: Host/Server, тип контента, клиент пользователя,
размер тела, язык, дата, кэширование.

3. Пустая строка
Указывает, что мета-информация отправлена и дальше будет тело сообщения.

4. Тело сообщения
Здесь могут быть содержание HTML-формы, HTML-документ и т.д.
Наличие тела и его размер определяется стартовой строкой и заголовками HTTP.
