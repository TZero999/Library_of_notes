
1) SELECT - столбцы, которые должны быть получены. 
Обязательное слово.

2) FROM - таблица для извлечения данных.
Обязательно при наличии SELECT.

3) WHERE - фильтрация на уровне строк.
Не обязательно.

4) GROUP BY - определение группы.
Только если выполняются итоговые вычисления по группам.

5) HAVING - фильтрация на уровне групп.
Не обязательно.

6) ORDER BY - порядок сортировки результатов.
Не обязательно.
