* URN 
Uniform Resource Name - неизменная последовательность символов, определяющая только имя ресурса.
* URL 
Uniform Resource Locator - уникальный адрес сайта в сети.
* URI
Uniform Resource Identifier - уникальный идентификатор, обобщённое понятие множества
идентификаций нашего ресурса. Включает в себя URN и URL (как вместе, так и по отдельности).
