- docker images
Проверяем наличие образов в системе

- docker ps
Проверяем запущенные контейнеры

- docker ps -a
Проверяем все существующие контейнеры

- docker build .
Создаём образ из файла "Dockerfile"

- docker run IMAGE_ID
Создаём контейнер из образа

- docker start CONTAINER_ID
Запускаем существующий контейнер

- docker container prune
Удаляет все контейнеры
